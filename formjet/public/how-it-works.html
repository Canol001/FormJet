How It Works

Learn how to set up FormJet and start sending emails in minutes.

üìã Step-by-Step Guide

  1. Create Your Form

    Create a form in your HTML with name, email, and message fields.

  2. Set the Form Action

    Set the action attribute to point to /send/yourUserId and method to POST.

  3. Form Submission

    Your users submit the form, and we send the data to your inbox.

  4. Configure Your Inbox

    Configure your recipient email in our dashboard or system backend.

üíª Code Example

<form action="/send/yourUserId" method="POST">
  <label for="name">Name</label>
  <input type="text" id="name" name="name" required>
  
  <label for="email">Email</label>
  <input type="email" id="email" name="email" required>
  
  <label for="message">Message</label>
  <textarea id="message" name="message" required></textarea>
  
  <button type="submit">Send</button>
</form>

üöÄ Getting Started

  1. Sign up for a FormJet account at signup page.
  2. Retrieve your unique userId from the dashboard.
  3. Configure your recipient email address in the dashboard settings.
  4. Add the form to your website using the code example above.
  5. Test your form by submitting a message and checking your inbox.
