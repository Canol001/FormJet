Getting Started

Welcome to FormJet! Follow these steps to set up your account and start sending emails.

  1. Sign up for a FormJet account using the "Sign up" button above.
  2. Create a new form from your dashboard by clicking "Add New".
  3. Copy the form endpoint URL provided in the dashboard.
  4. Integrate the endpoint into your website's form (see below).
  5. Start receiving submissions and sending emails!

Setting Up Your Form

To collect submissions, add the FormJet endpoint to your HTML form. Here's an example:

<!-- Add this form to your website -->
<form action="https://formjet.onrender.com/send/12345" method="POST">
  <label for="name"> Your name: </label>
   <input type="text" name="name" id="name">

  <label for="email"> Your email: </label>
   <input type="email" name="email" id="email">

  <label for="message"> Your message: </label>
    <textarea name="message" id="message"></textarea>
  <!-- your other form fields go here -->
  <button type="submit">Send</button>
</form>

Replace the `action` URL with the endpoint provided in your FormJet dashboard.

API Usage

FormJet provides an API for programmatically managing forms and submissions. Here's how to submit a form using the API:

// Example API request to submit a form
fetch('https://formjet.onrender.com/send/12345', {
method: 'POST',
headers: { 'Content-Type': 'application/json' },
body: JSON.stringify({
name: 'owano canol',
email: 'canolowana6@gmail.com',
message: 'Hello, FormJet!'
})
})
.then(response => response.json())
.then(data => console.log(data));

For more details on the API, including authentication and endpoints, visit the FormJet API documentation.

Troubleshooting

Encountering issues? Here are some common problems and solutions:

  • Form not submitting: Ensure the `action` URL matches your FormJet endpoint and the `method` is set to `POST`.
  • Spam submissions: FormJet automatically blocks spam using machine learning. If issues persist, contact support.
  • Email not received: Check your spam folder and ensure the recipient email is correctly configured in your form settings.
  • API errors: Verify your API key and endpoint. Refer to the API documentation for error codes.
